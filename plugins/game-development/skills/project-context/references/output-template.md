## 4. 코딩 스타일 & 네이밍 규칙

### 4.1 네이밍 규칙

| 대상 | 규칙 | 예시 ✅ | 예시 ❌ |
|------|------|---------|---------|
| 클래스 | PascalCase | `PlayerPresenter` | `playerPresenter` |
| 인터페이스 | `I` + PascalCase | `IPlayerView` | `PlayerView` (인터페이스) |
| 메서드 | PascalCase + 동사 시작 | `GetScore()` | `score()` |
| 비동기 메서드 | `{동사}Async` 접미사 | `LoadAsync()` | `Load()` (비동기인데) |
| private 필드 | `_camelCase` | `_health` | `health`, `m_health` |
| public 프로퍼티 | PascalCase | `MaxHealth` | `maxHealth` |
| {관찰된 항목} | {규칙} | {예시} | |

### 4.2 코드 스타일

